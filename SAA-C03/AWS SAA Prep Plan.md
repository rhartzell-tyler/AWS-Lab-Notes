# AWS SAA‑C03 Study Plan (Mid‑April Target)

## Overview
- Daily time: 2–3 hours  
- Duration: ~12 weeks  
- Exam target: Mid‑April 2026  
- Resources:
  - Pluralsight SAA course (video + labs)
  - AWS documentation + FAQs
  - Practice exams (Whizlabs, Tutorials Dojo, AWS Official)
  - Hands‑on labs in your AWS sandbox

---

# Week‑by‑Week Breakdown

## Weeks 1–2: Foundations Refresh

### Topics
- IAM fundamentals  
- VPC basics (subnets, routing, NAT, SG vs NACL)  
- EC2 basics (types, pricing, ASG intro)  
- S3 basics (storage classes, lifecycle, encryption)

### Actions
- Restart Pluralsight course  
- Take structured notes  
- Labs:
  - Build a VPC from scratch  
  - EC2 + ASG + ALB  
  - S3 lifecycle rules  

### Goal
- 30–40% course completion  
- Core AWS concepts refreshed  

---

## Weeks 3–4: Networking + Compute Deep Dive

### Topics
- VPC advanced (peering, TGW, endpoints, PrivateLink)  
- Load balancers (ALB vs NLB vs GWLB)  
- Autoscaling strategies  
- ECS, EKS, Elastic Beanstalk  
- Lambda + API Gateway patterns  

### Actions
- Continue Pluralsight  
- Labs:
  - VPC with endpoints  
  - ALB → ECS service  
  - Lambda + API Gateway  

### Goal
- 60–70% course completion  
- Networking feels intuitive  

---

## Weeks 5–6: Storage + Databases

### Topics
- RDS (multi‑AZ, replicas, failover)  
- Aurora (serverless v2, global DB)  
- DynamoDB (capacity modes, GSIs, DAX)  
- EFS vs FSx  
- S3 advanced (replication, events)

### Actions
- Pluralsight modules  
- Labs:
  - RDS failover  
  - DynamoDB autoscaling  
  - S3 replication + event triggers  

### Goal
- 80–85% course completion  
- Clear decision‑making between storage/database services  

---

## Weeks 7–8: Security + Monitoring + Serverless

### Topics
- KMS + encryption patterns  
- CloudTrail, Config, GuardDuty, Security Hub  
- IAM best practices  
- Lambda event sources  
- Step Functions  

### Actions
- Finish Pluralsight  
- Labs:
  - KMS CMK + envelope encryption  
  - CloudWatch dashboards + alarms  
  - Step Functions workflow  

### Goal
- Course complete  
- Strong AWS security architecture understanding  

---

## Weeks 9–10: Practice Exams + Weak Areas

### Actions
- Take 1 full practice exam

### Goal
- 75–80% on practice sets  
- Weak areas shrinking

### Actions
- 2–3 full practice exams  
- Review all incorrect answers  
- Re‑read AWS FAQs for:
  - S3  
  - EC2  
  - RDS  
  - DynamoDB

- Light labs only as needed  

### Goal
